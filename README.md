# AI Puzzle Solvers

This repository contains two artificial intelligence projects developed as part of my academic and personal practice in problem-solving and algorithm design.

## Projects Included

### 1. 8-Puzzle Solver
A solver for the classic 8-puzzle problem using multiple AI search strategies.  
The project focuses on comparing different methods based on performance, explored states, and solution efficiency.

### 2. 9x9 Sudoku Solver
A Sudoku solver that applies different solving approaches to complete standard 9x9 Sudoku boards.  
The project includes analysis and comparison of the implemented methods.

## Objectives
- Practice AI problem-solving techniques
- Implement and compare multiple solving algorithms
- Analyze algorithm efficiency and behavior on puzzle-based problems
